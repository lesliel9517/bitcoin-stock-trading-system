# Dashboard功能完整说明

## 已实现的所有功能

### 1. 价格统计区（顶部左侧panel）✓
显示完整的市场数据：
- **顶部大号价格**：当前价格 + 涨跌金额 + 涨跌百分比
- **第一列**：
  - 24H最高
  - 今开
  - 成交量
  - 历史最高
  - 历史最低
- **第二列**：
  - 24H最低
  - 昨收
  - 24H涨跌
  - 振幅

### 2. 策略显示与切换 ✓
- **顶部标题栏**：实时显示当前使用的策略
  - "均线交叉" (ma_cross)
  - "自适应策略" (adaptive_strategy)
- **快捷键 S**：按 `S` 键在两个策略之间切换
- **实时切换**：无需重启程序，策略立即生效

### 3. 图表区增强 ✓
- 蓝色价格线（●）
- 橙色均线（·）
- 白色当前价虚线（─）
- 左侧价格刻度（$格式）
- 右侧涨跌幅刻度（绿色/红色%）
- 买卖标记（B/S）

### 4. 周期切换 ✓
按数字键切换不同时间周期：
- `1` = 1D（实时数据）
- `2` = 5日
- `3` = 日K
- `4` = 周K
- `5` = 月K
- `6` = 年K

## 快捷键总览

```
Ctrl+C  = 退出程序
1-6     = 切换时间周期
S       = 切换策略
R       = 刷新显示
```

## 使用方法

### 启动交易
```bash
btc-trade trade start --strategy ma_cross --symbol BTC-USD --mode paper --dashboard
```

### 运行时操作
1. **查看当前策略**：顶部标题栏显示当前策略名称
2. **切换策略**：按 `S` 键，策略会在"均线交叉"和"自适应策略"之间切换
3. **切换周期**：按 `1-6` 数字键切换不同时间周期
4. **观察数据**：等待1-2秒，所有24h统计数据会自动更新

## 策略说明

### 均线交叉策略 (ma_cross)
- 简单的移动平均线交叉策略
- 短期均线上穿长期均线 → 买入信号
- 短期均线下穿长期均线 → 卖出信号
- 适合趋势明显的市场

### 自适应策略 (adaptive_strategy)
- 根据市场状态自动调整参数
- 检测市场趋势（上涨/下跌/震荡）
- 监控波动率（高/正常/低）
- 动态调整止损、止盈和仓位大小
- 适合多变的市场环境

## 数据更新说明

Dashboard启动后的数据流：
1. **0-1秒**：初始化，部分字段显示为 $0.00
2. **1-2秒**：接收Binance ticker事件，所有24h统计数据更新
3. **持续更新**：
   - Trade事件（高频）：更新当前价格
   - Ticker事件（每秒）：更新24h统计数据

## 故障排除

### 如果看不到数据
1. 停止程序（Ctrl+C）
2. 删除缓存：`find . -type d -name "__pycache__" -exec rm -rf {} +`
3. 重新运行程序
4. 等待2-3秒观察数据更新

### 如果无法切换策略
1. 确认按的是小写或大写 `s` 或 `S`
2. 查看日志区是否显示"切换策略"消息
3. 查看顶部标题栏策略名称是否改变

### 如果历史K线加载失败
配置代理：
```bash
export HTTP_PROXY=http://your-proxy:port
export HTTPS_PROXY=http://your-proxy:port
```

## 技术细节

### 策略切换实现
- 按 `S` 键触发 `dashboard.switch_strategy()`
- 停止当前策略实例
- 创建新策略实例并启动
- 保留所有历史数据和持仓状态
- 新策略立即开始接收市场数据

### 数据持久化
- 策略切换不影响持仓和资金
- 交易历史完整保留
- 图表数据继续累积
